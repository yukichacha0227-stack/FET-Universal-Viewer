# Sample Data

This directory contains example `.Dat` files used for local testing and demonstration.

The files follow the expected FET measurement format:

- `No.`
- `Temp`
- `Mag`
- `Isd`
- `Vsd`
- `Vbg`

Japanese headers such as `温度` and `磁場` are normalized to `Temp` and `Mag` by the data loader.
